    如何更新配置数据

    由于浏览器安全限制，网页无法直接读取系统文件。要更新配置数据：

    1. 在 Claude Code 中说 "更新配置数据"
    2. Claude 会扫描你的 ~/.claude/ 目录
    3. 生成新的 config-data.js 文件
    4. 刷新本页面即可看到最新数据
